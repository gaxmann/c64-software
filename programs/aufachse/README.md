# „Auf Achse, Computerspieler“ für C64

Dieses Programm ist ein Computer-Spieler (KI) für das Brettspiel „Auf Achse“. Es wird am Spieltisch parallel zum Brettspiel verwendet: Ein menschlicher Spieler würfelt und zieht den grauen Lkw, das Programm trifft die Entscheidungen (belädt, bietet mit, plant Routen). Es ist relativ stark (zumindest in der Regelfassung von 1987), aber noch schlagbar.
