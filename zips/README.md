# ZIP Downloads

This folder contains ready-to-download ZIP packages.

## How to download a ZIP file on the GitHub website (recommended)

Open this folder (`/zips`) in your browser.
1. Click the ZIP file you want (for example `Anabasis_en.zip`).
2. On the file page, use one of these options:
   - Click **Download** (if shown), or
   - Click **View raw** (or the **â€¦** menu and then **Download** / **View raw**, depending on the GitHub layout).
3. Your browser will start downloading the ZIP. You can also copy the URL from the address bar to share a direct download link.
